# RFM Analysis

This project performs RFM (Recency, Frequency, Monetary) analysis for customer segmentation and reporting.

## Project Structure
- `rfm_analysis.py`: Main script for RFM analysis.
- `config.json`: Configuration file for analysis parameters.
- `requirements.txt`: Python dependencies.
- `.gitignore`: Files and folders to be ignored by git.

## Usage
1. Run the analysis:
   ```bash
   python rfm_analysis.py
   ```

## Output
- Analysis results are saved to `rfm_analysis_result.xlsx`.
